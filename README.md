# Astragalus-Sheep-GMM-3D
This repository is linked to the article "Exploring biological and ecological components of sheep astragalus size and shape variation using 3D geometric morphometrics: towards a bioarchaeological proxy" - Vuillien et al., 2026 - DOI : 

The supplementary data 2 and 3 corresponded to :
Supplementary data 2: R Script automation produced by P. Garberi and M. Vuillien and its example dataset, which contains the coordinates of the first five specimens from the dataset used in the article. 
Supplementary data 3: R script for the statistical analyses associated with the files containing three-dimensional benchmarks that describe the morphological patterns of each specimen studied, as well as the 3D model of the mean astragalus that was used to project the deformations of the first two axes of the PCA.
